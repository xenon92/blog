Crasher

7th July, 2014

Open source .NET based GUI for a CLI hacking code that exploits a vulnerability in old gaming servers, causing them to crash.

  • This GUI requires the CLI code for it to work. Without that code, this GUI is useless
  • DISCLAIMER is added with the GUI. READ IT!
  • I neither the endorse the hacking code, nor will I include it along with the source code of the GUI
  • This GUI was made solely for educational purposes

Screenshot

Features

  • Server name is IP independent
  • Ability to save and delete the servers with their name, IP and port
  • System balloon notifications
  • Very low on system memory

System Requirements

  • .NET Framework
  • Windows XP, Windows Vista, Windows 7, Windows 8, Windows 8.1
